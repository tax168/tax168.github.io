欢迎来到我的个人博客！🎉

这是使用 Hugo 和 PaperMod 主题创建的第一篇示例文章。通过这篇文章，我将向大家展示如何使用这个强大的静态网站生成器。

为什么选择 Hugo？#

Hugo 是一个非常快速的静态网站生成器，具有以下优势：

1. 极快的构建速度#

相比其他静态网站生成器，Hugo 的构建速度非常快：

1本地启动开发服务器
2hugo server -D
3构建生产版本
4hugo --minify

2. 丰富的主题生态#

PaperMod 是我选择的主题，它有以下特点：

  • 🚀 快速加载
  • 🌙 支持深色模式
  • 📱 响应式设计
  • 🔍 内置搜索功能
  • 📝 简洁优雅的设计

PaperMod 主题特性#

安装方式#

进入你的 Hugo 站点目录

1cd your-site

添加 PaperMod 主题作为子模块

1git submodule add https://github.com/adityatelange/hugo-PaperMod.git themes/hugo-PaperMod

配置要点#

在 hugo.toml 中配置主题：

1baseURL = "https://your-domain.com/" 
2languageCode = "zh-cn" 
3title = "我的博客" 
4theme = "hugo-PaperMod"
5[params] 
6defaultTheme = "auto" 
7ShowShareButtons = true 
8ShowReadingTime = true

文章内容组织#

Front Matter（前置参数）#

每篇文章的开头都包含 YAML 格式的 Front Matter，用于定义文章的元数据：

1title: "文章标题" 
2description: "文章描述" 
3date: 2026-03-31 
4categories: ["分类 1", "分类 2"] 
5tags: ["标签 1", "标签 2"]

使用 <!--more--> 控制摘要#

在文章中使用 <!--more--> 标记来控制首页显示的摘要内容。标记之前的内容会显示在文章列表中。
