语雀知识库下载器 - 使用指南#

安装#

双击运行 语雀知识库下载器 Setup x.x.x.exe，按提示选择安装目录，完成安装后桌面会生成快捷方式。

第一步：激活软件#

首次打开软件会显示激活界面：

  1. 界面上会自动显示你的机器码（每台电脑唯一）
  2. 点击「复制」按钮，将机器码发送给管理员
  3. 管理员会回复一串授权码
  4. 将授权码粘贴到输入框，点击「激活」

激活成功后会自动进入主界面。授权信息会保存在本地，下次打开无需重复激活。

注意：授权码与电脑硬件绑定，更换电脑后需要重新获取授权码。

第二步：下载知识库#

基本使用#

  1. 打开语雀，进入你要下载的知识库页面，复制浏览器地址栏的 URL
    • 格式类似：https://www.yuque.com/用户名/知识库名
  2. 将 URL 粘贴到软件的「知识库 URL」输入框
  3. 点击「选择目录」选择文件保存位置（默认保存到"文档/yuque-download"）
  4. 点击「开始下载」

下载过程中可以在进度条和日志区域查看实时状态。完成后点击「打开目录」可直接在资源管理器中查看下载的文件。

下载私有知识库#

如果知识库需要登录才能访问，需要填写 Token：

  1. 在浏览器中登录语雀
  2. 按 F12 打开开发者工具，切换到「应用」(Application) 标签
  3. 左侧找到 Cookie → https://www.yuque.com
  4. 找到名为 _yuque_session 的条目，复制它的值
  5. 在软件中展开「🔐 认证设置」，将值粘贴到 Token 输入框

如果是企业版语雀，Cookie 名称可能不是 _yuque_session，请在 Cookie Key 中填写实际的名称。

高级选项说明#

展开「⚙️ 高级选项」可以看到以下设置：

选项说明
忽略图片不下载文档中的图片，仅保留文字内容
忽略附件不下载文档中的附件文件
生成文档 TOC在每篇文档中生成目录索引
增量下载只下载新增或更新的文档，适合重复下载同一知识库
视频链接转 video 标签将视频链接转为 HTML video 标签，方便本地播放
隐藏页脚信息不在文档末尾添加更新时间和原文链接

第三步：Web 预览（可选）#

下载完成后，可以通过内置预览服务在浏览器中阅读知识库：

  1. 切换到「🌐 预览」标签页
  2. 点击「选择目录」，选择下载生成的知识库文件夹（包含 index.md 的目录）
  3. 点击「启动预览」，浏览器会自动打开预览页面
  4. 左侧为文件夹树形导航，点击文档名即可阅读，支持代码高亮
  5. 使用完毕后点击「停止」关闭预览服务，或直接关闭软件会自动停止

预览服务运行在本地 18888 端口，仅本机可访问，不会暴露到网络。如果端口被占用会自动尝试下一个端口。

常见问题#

Q: 下载失败，提示"未找到知识库 ID" A: 请检查 URL 是否正确，确保是知识库地址而不是单篇文档地址。

Q: 下载失败，提示"知识库目录为空" A: 知识库可能需要登录才能访问，请在「认证设置」中填写 Token。

Q: 部分文档下载失败 A: 画板类型和数据表类型的文档暂不支持下载，日志中会显示具体的失败原因。

Q: 预览启动失败 A: 请确认选择的是包含 .md 文件的知识库目录。如果提示端口占用，服务会自动尝试下一个端口。

Q: 换了电脑后软件提示需要重新激活 A: 授权码与硬件绑定，请将新电脑的机器码发送给管理员获取新的授权码。
